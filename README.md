# 🎬 Movie Recommendation System

This is a Machine Learning project that recommends movies based on similarity.

## 🚀 Features

* Content-based recommendation system
* Built using NLP
* Interactive UI using Streamlit

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* NLTK
* Streamlit

## ⚙️ How to Run

1. Clone the repo:
   git clone https://github.com/YOUR_USERNAME/Movie_Recommendation_System.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

## ⚠️ Note

`.pkl` files are not uploaded due to size limit.
Run notebook to generate them.
